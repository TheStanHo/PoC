# Petit Crumb Cafe PoC

A soft, modern, minimal bakery cafe website concept for a fictional neighbourhood shop.

It uses synthetic cafe details and public Unsplash image URLs. No real business data, private imagery, or confidential details are included.

## Current Scope

- Full-bleed hero with brand-forward presentation.
- About section with cafe story and interior photography.
- Interactive menu with category tabs, featured items, and prices.
- Visit section with location, contact, and opening hours.
- Cutesy modern visual direction with soft sage and blush tones.

## Future Ideas

- Add nested pages for seasonal menus under `/pocs/petit-crumb/menu`.
- Add a simple reservation or catering enquiry form.
- Add an allergy / dietary notes panel for menu items.
