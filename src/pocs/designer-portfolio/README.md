# Designer Portfolio PoC

This PoC is a simple website design concept for a fictional graphic designer portfolio.

It uses synthetic project names and public Unsplash image URLs. No client work, private imagery, or confidential details are included.

## Current Scope

- Portfolio hero section.
- Featured project cards.
- Services section.
- Simple contact call to action.
- Dark creative styling that still fits the PoC hub.

## Future Ideas

- Add case-study detail pages under `/pocs/designer-portfolio/projects/<project-id>`.
- Add project category filtering.
- Add a lightweight image gallery.
- Add a CMS-backed content model for real portfolio updates.
