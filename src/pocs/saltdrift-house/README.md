# Saltdrift House PoC

A coastal linen boutique hotel and spa landing page for a fictional headland retreat.

It uses synthetic hotel details and public Unsplash image URLs. No real business data, private imagery, or confidential details are included.

## Current Scope

- Full-bleed hero with brand-forward presentation.
- Interactive room selector with suite details and photography.
- Spa section with a short ritual list.
- Coast story section with shoreline imagery.
- Visit section with location, contact, and enquiry call to action.
- Coastal linen visual direction with sea-mist and oat tones.

## Future Ideas

- Add nested pages for rooms under `/pocs/saltdrift-house/rooms`.
- Add a simple stay enquiry form with date fields.
- Add a spa booking calendar with mock availability.
